UI/UX Design • Landing Page & Web Experience

SoulfulTone

A calming and emotionally driven website designed to introduce an affirmation app, focusing on minimalism, soft visuals, and peaceful user experience to create a sense of mental comfort and clarity.

Overview
Problem Statement

Many mental wellness platforms feel either too clinical or overly complex. Users looking for emotional support need a calm, distraction-free environment that feels safe, simple, and soothing.

Solution

Designed a soulful, minimalist web experience using soft pastel colors, clean typography, and gentle spacing to create a peaceful digital environment that encourages users to explore affirmations effortlessly.

Execution
Key Features
  • Emotion-driven Hero Section with calming visuals
  • Minimal Navigation System for clarity and focus
  • Soft UI Feature Cards explaining app benefits
  • Clean Login & Subscription Flow
  • Informational Blog & About Pages
  • Fully Responsive Design (Mobile-first approach)
Design Style
  • Tone: Soulful, calming, warm
  • Colors: Soft pastels & light gradients
  • Typography: Modern, rounded, clean
  • UI Elements: Rounded edges, airy spacing, minimal shadows
My Role (UX Designer)
  • Created full user flow
  • Designed wireframes & UI screens
  • Defined visual direction and emotional tone
  • Focused on simplicity & user comfort
Tools Used

Figma

Outcome

A visually soothing and structured web experience that balances emotional storytelling with usability, making the affirmation app feel approachable and peaceful.

Important Insight

This project is not just UI—it shows emotional UX design, highlighting the ability to create environments that evoke a specific feeling and comfort, going beyond standard visual interfaces.
